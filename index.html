<app-store>, <app-form>, <app-todos>

This is an example of the web component created with the hybrids library.

It shows how to use parent factory to create simple application with shared state between custom elements. 

<app-store todos='["One", "Two"]'>
  <app-todos></app-todos>
  <app-form></app-form>
</app-store>

Use form to add todo to the list. Click on todo on list to remove it. Change default todos in <app-store> element in developer tools.
